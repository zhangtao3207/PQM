## 当前接口与配置

- 当前使用串行接口模式，读取时序按 SPI 风格处理。
- 当前 `PAR/SER/BYTE_SEL = 1`，`DB15/BYTE_SEL = 0`，因此工作在串行模式。
- 当前 `RANGE = 0`，量程为 `±5V`。
- 当前 `REF_SELECT = 1`，使用内部 2.5V 基准。
- 当前 `CONVSTA` 与 `CONVSTB` 已短接，共用 `FPGA_CONVST`。
- 当前实际使用通道：
  - `V1 -> ADC_A_IN`，电流采样输入
  - `V2 -> ADC_V_IN`，电压采样输入
  - `V3 ~ V8` 未使用

## 当前 FPGA 相关信号

### FPGA 输出信号

- `FPGA_CONVST`
- `FPGA_RESET`
- `FPGA_SCLK`
- `FPGA_CS#`

### FPGA 输入信号

- `FPGA_FRSTDATA`
- `FPGA_BUSY`
- `FPGA_DOUTA`
- `FPGA_DOUTB`

## 当前时序流程

1. 上电。
2. `SCLK` 取 `6.25 MHz`，对应 `50 MHz` 主频的 8 分频。
3. 复位芯片：`RESET` 拉高大于 `50 ns`，再拉低。
4. 启动采样：对 `CONVST` 给上升沿。
5. `BUSY = 1`：芯片正在转换。
6. 等待 `BUSY` 下降沿，表示转换完成。
7. `CS` 拉低，开始串行输出。
8. 首帧输出时，`FRSTDATA = 1`，对应 `V1` 的 `MSB`。
9. 连续提供 `SCLK`，每 `16` 个时钟读取 1 路数据。

### 串行读取说明

- 单线 `DOUTA` 读取：
  - `V1 -> V2 -> V3 -> ...`
  - 共 `8 组 x 16 bit`
- 双线 `DOUTA / DOUTB` 读取：
  - 同时读取两组数据
  - 共 `4 组 x 16 bit`

10. `CS` 拉高，本次读数结束。
11. 等待下次采样。

## 备注

- 当前文档按板上现有接法整理，重点用于 FPGA 串行采样联调。
- 当前项目主要使用 `V1` 和 `V2` 两路。
