XC7Z020-2CLG400I 核心页实施方案 第三步

目标
本文件把核心原理图推进到可直接开始绘制
本阶段只冻结 Sheet 01 到 Sheet 04
本阶段仍不冻结 DDR 器件 Boot 媒介和 MIO 外设

U1 原理图拆分方式
不要把 U1 画成一个大符号
第一版建议按功能拆成多个单元

+----------------------+------------------------------------------+--------------------------------------+
| 单元名               | 包含内容                                 | 目的                                 |
+----------------------+------------------------------------------+--------------------------------------+
| U1_PS_PWR            | VCCPINT VCCPAUX VCCPLL VCCO_MIO0 VCCO_MIO1 VCCO_DDR | 电源页集中管理              |
| U1_PL_PWR            | VCCINT VCCBRAM VCCAUX VCCO_xx VCCADC_0 GNDADC_0      | 电源页集中管理              |
| U1_RESET_CFG         | PS_POR_B PS_SRST_B PS_CLK DONE INIT PROGRAM JTAG     | 时钟复位配置集中            |
| U1_DDR               | 全部 PS_DDR 专用脚                        | 暂不连器件但先占位                   |
| U1_MIO               | PS_MIO[53:0] PS_MIO_VREF                  | 暂不分配外设                         |
| U1_PL_BANK13         | CLG400 对应 Bank 13 用户 IO               | 后续谨慎使用                         |
| U1_PL_BANK34         | CLG400 对应 Bank 34 用户 IO               | 后续主要接口候选                     |
| U1_PL_BANK35         | CLG400 对应 Bank 35 用户 IO               | 后续主要接口候选                     |
| U1_XADC              | VP VN VREFP VREFN AD0P/N 到 AD15P/N       | 若不用也先保留规则                   |
+----------------------+------------------------------------------+--------------------------------------+

Sheet 01 Power Tree 实施规则

+----------------------+------------------------------------------+--------------------------------------+
| 模块                 | 实施要求                                 | 备注                                 |
+----------------------+------------------------------------------+--------------------------------------+
| VIN_PRE_REG          | 板级预稳压输入占位                       | 本阶段不冻结外部输入接口             |
| REG_PS_1V0           | 生成 PS_1V0                              | 独立 rail                            |
| REG_PL_1V0           | 生成 PL_1V0                              | 与 VCCBRAM 同源同网                  |
| REG_PS_1V8           | 生成 PS_1V8                              | 独立 rail                            |
| REG_PL_1V8           | 生成 PL_1V8                              | 独立 rail                            |
| FB_PLL               | PS_1V8 到 PS_PLL_1V8                     | 120 欧姆 at 100 MHz                  |
| REG_MIO0_VCCO        | 生成 MIO0_VCCO                           | 保持独立                             |
| REG_MIO1_VCCO        | 生成 MIO1_VCCO                           | 保持独立                             |
| REG_DDR_VCCO         | 生成 DDR_VCCO                            | 保持独立                             |
| REG_XADC_1V8         | 可与 PL_1V8 同源 但逻辑上独立            | 保持模拟域独立性                     |
| SUP_POR              | 监控 PS_1V0 PS_1V8 MIO0_VCCO             | 输出控制 PS_POR_B                    |
| PG_SUMMARY           | 汇总各 rail 的 PGOOD                     | 便于系统调试                         |

Sheet 02 U1 Power 实施规则

+----------------------+------------------------------------------+--------------------------------------+
| 项目                 | 实施要求                                 | 说明                                 |
+----------------------+------------------------------------------+--------------------------------------+
| 电源脚分组           | 按 rail 分组连接                         | 不要把所有电源脚散落在多页           |

| VCCADC_0             | 单独出现在 U1_PL_PWR 或 U1_XADC 单元     | 方便后续模拟布局                     |

本阶段不要做的事

+----------------------+------------------------------------------+--------------------------------------+
| 禁止项               | 原因                                     | 处理方式                             |
+----------------------+------------------------------------------+--------------------------------------+
| 冻结 Boot Mode 电阻  | 缺少 UG585 支撑                          | 留空位 等下一步                      |
| 冻结 QSPI 器件       | 缺少 MIO 分配                            | 留到下一步                           |
| 冻结 DDR 颗粒        | 缺少 DDR 标准与容量                      | 留到下一步                           |
| 合并 PS_1V0 与 PL_1V0| 会隐藏噪声边界                           | 保持独立 rail                        |
| 合并 MIO0 与 MIO1    | 会锁死未来外设电平                       | 保持独立 rail                        |
| 只靠 RC 产生 POR     | 时序不可控                               | 用监控器                             |
